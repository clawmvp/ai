# cursor-ai

Project inițializat cu Git, legat de [github.com/clawmvp/ai](https://github.com/clawmvp/ai).

---

## Implicare minimă (un singur pas)

După ce ai instalat **Xcode Command Line Tools** (dacă a apărut fereastra, apasă *Install* și așteaptă; altfel deschide Terminal și rulează `xcode-select --install`), copiază și rulează în **Terminal**:

```bash
cd /Users/claudiunae/cursor-ai && bash setup-github.sh
```

La `git push` ți se va cere autentificare GitHub (utilizator + parolă sau [Personal Access Token](https://github.com/settings/tokens) dacă ai 2FA). Introdu și apasă Enter — gata.
